2. What is xnewsletter?
 _____________________________________________________________________

 newsletter module for XOOPS

 3. Requirements
 _____________________________________________________________________

 - PHP version >= 5.3.7
 - XOOPS >= 2.5.7
 - ModuleClasses in /Frameworks (download it from here: http://goo.gl/Bmknt)


 4. Installation
 _____________________________________________________________________

 Simply upload the xnewsletter module folder to the /modules directory of your
 XOOPS site and install it via XOOPS modules administration panel. You can
 configure module settings through xnewsletter module control panel.
 Detailed instructions on installing modules are available in the
 XOOPS Operations Manual (http://goo.gl/adT2i)
